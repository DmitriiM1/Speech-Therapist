# Speech Therapy Center Website

This project is a modern informational website for a Speech Therapy Center.  
It was built with **React**, **Bootstrap**, and custom UI components.

## Live Demo

You can view the live version of the website here:

**https://dmitriim1.github.io/Speech-Therapist/**

## Project Overview

The website provides:

- Information about speech therapy services  
- Profiles of specialists  
- Pricing details  
- Useful materials for parents  
- News and updates  
- Customer reviews  
- Contact information  
- Online application form  

The goal of the project is to create a clean, user-friendly, and responsive web platform for parents and clients looking for professional speech therapy services.

## Technologies Used

- React (Create React App)
- React Router
- Bootstrap
- React Icons
- Formik + Yup for form validation
- Axios
- Custom CSS styling

## Features

- Fully responsive design  
- Smooth navigation between sections  
- Interactive components  
- Image carousel  
- Contact form with validation  
- Preloader animation  
- Modular component structure
